How to install:
1. Add openupm scope to manifest.json
"scopedRegistries": [
    {
      "name": "OpenUPM",
      "url": "https://package.openupm.com",
      "scopes": [
        "com.cysharp.unitask"
      ]
    }
  ]

2. add beton to UPM with link
  https://github.com/vovaBuddy/beton.git?path=Beton#develop
